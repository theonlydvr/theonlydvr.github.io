SnowDLAke

This is a snowflake simulator inspired by a Code Golf post based on diffusion limited aggregation (DLA). You can choose the symmetry (3-12 points), number of particles used to generate each section of the flake, how emphasized the diagonals are, the size of the flake, how large the points in the flake are, and the flake colors. Parameters are summarized in the seed which you can enter to recreate a particular flake (press the arrow to enter a seed). The snowflake button will draw your flake with the current parameters and random seed. The reset button will draw a flake from a new random seed while keeping the current parameters. Enjoy the snow!
