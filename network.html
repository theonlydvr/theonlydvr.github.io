Network

This is a network simulation I developed for the website banner to give a feeling similar to a mass of connected, shifting neurons. It is originally generated by a random walk with a biasing term in two directions with some probability of branch creation and termination at each step. The network is simplified by computing its minimum spanning tree (MST) before visualization making connections a little more neat. Creating an MST this large is a relatively intensive computation which can be significantly sped up by computing the Delauney triangulation as a preprocessing step.
